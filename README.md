# 🌍 Travel Linker – Travel Services Management Platform

Travel Linker is a multi-platform system designed to connect travel service providers with customers through a unified management platform. The project consists of a web dashboard and a mobile application, supported by a centralized backend.

---

## 👥 User Roles

### Customer

* Browse hotels and transportation services
* Book rooms and trips
* View booking history

### Enterprise (Hotel / Transport Company)

* Manage services, rooms, and vehicles
* Schedule trips and room availability with an integrated scheduling system to maximize resource utilization
* Track bookings and reservations
* Access dashboard for analytics and management

### Admin

* Manage users and enterprises
* Oversee all bookings and activities
* Approve enterprise accounts
* Handle system-wide settings

---

## 🚀 Features

* Hotel management (CRUD for rooms and availability)
* Transportation management (CRUD for trips and vehicles)
* Scheduling system for rooms and vehicles to optimize usage
* Booking system with real-time availability
* RESTful API for mobile app
* Authentication and authorization
* Role-based access control

---

## 🧑‍💻 Technologies Used

* ASP.NET Core MVC
* ASP.NET Core Web API
* Entity Framework Core
* MS SQL Server
* LINQ
* HTML, CSS, Bootstrap

---

## 🎯 Project Goal

The goal of Travel Linker is to provide a scalable and flexible platform that simplifies travel service management, unifies hotel and transportation bookings, and supports both web and mobile clients. The integrated scheduling system ensures efficient utilization of rooms and vehicles.
